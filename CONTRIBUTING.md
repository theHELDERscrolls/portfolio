# Contributing

Thanks for your interest in this project!

This repository contains my personal portfolio.  
While this project is primarily personal, feedback and suggestions are always welcome.

## How to contribute

- Feel free to open an **issue** if you find a bug or have a suggestion.
- If you want to submit a **pull request**, please open an issue first to discuss the proposed changes.

## Development setup

```bash
npm install
npm run dev
```

## Code style

- Follow the existing project structure.
- Use TypeScript and keep types strict.
- Prefer small, focused components.
