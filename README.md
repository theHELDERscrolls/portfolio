# Personal Portfolio

Personal portfolio built with React, TypeScript and Vite, focused on clean architecture, reusable components and a smooth user experience.

The project showcases my background, experience, projects and technical stack, along with small interactive features powered by third-party APIs.

## Features

- Responsive layout (desktop & mobile)
- IP-based geolocation
- Real-time weather information
- Dynamic quotes integration
- Modular and scalable project structure
- Smooth animations and visual effects

## Tech Stack

- React
- TypeScript
- Vite
- React Router
- Axios
- Tailwind CSS
- GSAP

## Development Setup

```bash
# Install dependencies
npm install

#Run the development server
npm run dev
```

Environment variables and API keys required for development are documented in [CONTRIBUTING.md](./CONTRIBUTING.md).
